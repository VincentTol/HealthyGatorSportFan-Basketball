Steps to send notifications to local Development app
Run frontend and backend
If frontend terminal says "Using Development build" press S to switch to expo go
Install expo go app on phone, and login
Scan QR code with phone camera, select open in expo go
Go to the frontend terminal where you ran the frontend
Look for 'push_token': 'ExponentPushToken[]' and copy what is between the []
Replacing YOUR_COPIED_TOKEN below with the value you copied earlier
Find the user id next to that push_token value
Copy that as well, and paste below where it says 'user_id'
Open new terminal in Django folder
- venv\Scripts\activate
- python manage.py shell
